# Portfolio Website

A modern, responsive **personal portfolio website** showcasing projects, skills, creative work, and contact information. Built with **HTML, CSS, JavaScript**, and enhanced with animations using **GSAP**.

---

## Features

- Hero section with **typing animation** and profile image  
- Showcase of **skills** with icons and hover effects  
- Dynamic **projects section** with filtering and animations  
- **Design & photography gallery** with category tabs  
- **Educational background** section  
- Contact form with basic validation  
- Light/Dark mode toggle with **localStorage support**  
- Smooth scrolling and responsive navigation  

---

## Technologies Used

- HTML5 & CSS3  
- JavaScript (ES6+)  
- GSAP for scroll and animation effects  
- Font Awesome for icons  
- Google Fonts for typography
